Pros and Cons of .pngs and .jpgs

Pros of .pngs

  • Transparency is supported
  • Does not undergo lossy image compression, more detail can be stored in the file
  • Can support many more color options compared to other formats

Cons of .pngs

  • Cameras typically shoot photos in the .jpg image format
  • Difficult to transfer files because of the large file sizes
  • Supports CMYK color modes, printing from .png is therefore difficult
